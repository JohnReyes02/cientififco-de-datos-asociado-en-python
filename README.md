# Científico de Datos Asociado en Python

Este directorio alberga materiales relacionados con la formación de Científico de Datos Asociado. Actualmente se centra en modelos avanzados de Machine Learning.

## Contenido Actual

*   **Machine Learning con Modelos Basados en Árboles**: Enfoque en algoritmos de árboles de decisión, Random Forests y Boosting.

## Tecnologías

*   **Lenguaje**: Python
*   **Librerías**: Scikit-learn, Pandas, NumPy.

## Configuración del Entorno

1.  **Crear el entorno virtual**:
    ```bash
    python3 -m venv .venv
    ```
2.  **Activar el entorno**:
    *   macOS/Linux: `source .venv/bin/activate`
    *   Windows: `.venv\Scripts\activate`
3.  **Instalar dependencias** (si aplica):
    ```bash
    pip install -r requirements.txt
    ```
